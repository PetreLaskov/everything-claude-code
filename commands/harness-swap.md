---
description: Show available harnesses and how to switch between them
---

Print this menu for the user:

```
Available Harnesses:

  harness-swap v8                → V8 Meta-Architecture Harness Engine
  harness-swap ecc               → Everything Claude Code (dev-focused)
  harness-swap cyborgism-harness → Cyborgism Harness (Cognitive OS)
  harness-swap cyborgism         → Cyborgism 3.0
  harness-swap original          → Restore original .claude config
  harness-swap status            → Show which harness is active

How to switch:
  1. Exit Claude Code
  2. Run: harness-swap <name>
  3. Start a new session
```
